# Web3_FinalProject_LaravelBookstore

## Introduction
This is a Laravel-based bookstore website I created as the Final Project for Web 3, my final Web Development course in the 3-semester path I am taking at Florida Southern College. The application builds upon a bookstore project from earlier web development courses and extends it using Laravel and Herd.

## Features
- Browse books
- Add, edit, and delete inventor
- Schedule, delete, and see events and deals
- Create, manage, delete users
- User authentication
- Database-backed persistence

## Tech Stack
- Laravel
- PHP
- MySQL
- Laravel Herd
- Blade / Vite

- ## Setup
1. Clone the repository
2. Copy `.env.example` to `.env`
3. Configure database credentials
4. Run migrations
5. Start the local server
